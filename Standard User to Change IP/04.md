To get the current logged-in **username** in Windows via the command line, you can use any of these methods:

### **1. Using `%USERNAME%` (Environment Variable)**
```cmd
echo %USERNAME%
```
- **Output Example:**  
  ```
  SDE-OFC-001
  ```
- **Pros:** Fastest and simplest method.

### **2. Using `whoami` (Built-in Command)**
```cmd
whoami
```
- **Output Example:**  
  ```
  SDE-OFC-001\AdminUser
  ```
- **For just the username (without domain):**  
  ```cmd
  whoami | awk -F "\\" '{print $2}'
  ```
  *(Requires **Windows Subsystem for Linux (WSL)** or **Git Bash** for `awk`)*  
  *(Alternative without `awk`: `for /f "tokens=2 delims=\" %a in ('whoami') do @echo %a`)*

### **3. Using `net` (Older Windows Versions)**
```cmd
net config workstation | findstr "User name"
```
- **Output Example:**  
  ```
  User name                   SDE-OFC-001\AdminUser
  ```
- **Pros:** Works even in legacy Windows versions.

### **4. Using `wmic` (Advanced Query)**
```cmd
wmic computersystem get username
```
- **Output Example:**  
  ```
  UserName  
  SDE-OFC-001\AdminUser  
  ```
- **Pros:** Useful in scripts for precise control.

---

### **Which One Should You Use?**
- **For simplicity:** `echo %USERNAME%`  
- **For domain-aware usernames:** `whoami` or `wmic`  
- **For scripting:** `%USERNAME%` (fastest) or `wmic` (most reliable).
